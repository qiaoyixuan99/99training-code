# Training Code

个人编程训练项目合集。

## 项目目录

### 📧 email-projects
邮件追踪系统，基于 Flask 构建。包含 4 个迭代版本：
- **email-tracker-v1** — 初版邮件追踪器
- **email-tracker-v2** — 增强版本，引入数据库存储
- **email-tracker-v3** — 功能迭代优化
- **email-tracker-v4** — 最新版本，包含启动脚本 `启动邮件追踪V4.py`

### 🗺️ squad-maps
Squad 游戏战术地图工具，包含多张地图的瓦片数据和战术计算器。

### 📱 daily-forge-miniapp
微信小程序 — 每日锻造（daily-forge）。

### 📐 measure-timeline
时间线/进度测量工具。

### 📖 study-plan
学习计划与笔记。
