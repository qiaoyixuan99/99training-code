## ⚡ 使用流程

1. 打开 `商品文案.md`，选择要上架的商品
2. 对照 `03-内容制作指南/内容大纲.md` 制作对应的 PDF/资料
3. 复制标题和文案到闲鱼发布页
4. 按图片建议制作/寻找首图（可用 Canva 快速生成）
5. 发布后标记日期，等待出单

---

## 💡 标题优化公式

```
核心关键词 + 用户痛点 + 信任背书 + 稀缺感

例：
  ❌ "Python教程" （太泛，没人搜）
  ✅ "Python自动化脚本10个案例 | 办公重复工作一键搞定 | 亲测好用"
```

---

## 🔄 淘汰机制

- 上架满 7 天后，曝光 < 50 的品 → 直接下架
- 上架满 14 天，有曝光但零转化的品 → 优化标题/首图后重新上
- 持续出单的品 → 复制变体（换标题/首图/关键词再上一个）
