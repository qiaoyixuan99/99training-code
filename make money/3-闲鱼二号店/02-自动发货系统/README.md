# 02-自动发货系统

> 利用闲鱼内置自动回复 + Python 辅助脚本，实现 7×24 小时自动发货，真正躺着收钱。

---

## 🏗️ 系统架构

```
买家下单/咨询
    │
    ▼
闲鱼内置自动回复 ──→ 发送网盘链接（基础方案，推荐先用）
    │
    ▼
Python辅助脚本 ──→ 管理链接 + 统计发货 + 防失效监控（进阶方案）
```

---

## 📂 文件说明

| 文件 | 用途 | 必读 |
|------|------|------|
| `auto_reply.py` | 自动化辅助主程序 | ✅ |
| `templates.json` | 回复话术模板库 | ✅ |
| `requirements.txt` | Python 依赖包 | - |

---

## 🚀 方案一：闲鱼内置自动回复（零代码，推荐先用）

### 设置步骤
1. 打开闲鱼 App → 我的 → 右上角设置
2. 进入「消息设置」→「自动回复」
3. 设置关键词规则（见下方配置表）
4. 把网盘链接放在自动回复里

### 关键词规则配置

```
关键词：教程 | 链接 | 发货 | 怎么买 | 多少钱 | 拍了 | 下单
自动回复内容：
━━━━━━━━━━━━━━━━
📦 您好，感谢下单！

🔗 资料链接：[你的网盘链接]
📂 提取码：xxxx

如链接失效请留言，看到立刻补发 ❤️

拍下后麻烦给个好评，您的支持是我更新的动力 🙏
━━━━━━━━━━━━━━━━
```

### 不同商品的自动回复模板

具体话术见 `templates.json`，每个商品都有专属回复。

## 💻 方案二：Python 辅助脚本（进阶，需要电脑常开）

### 功能
1. **链接管理**：集中管理所有商品的网盘链接
2. **发货记录**：自动记录每笔发货
3. **链接监控**：定期检查网盘链接是否失效
4. **快捷回复生成**：选择商品 → 一键生成回复文案

### 安装

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 编辑 config（在 auto_reply.py 里修改网盘链接）
```

### 运行

```bash
# 交互模式：选择商品 → 生成回复文案
python auto_reply.py

# 链接检查模式：检查所有网盘链接是否有效
python auto_reply.py --check

# 统计模式：查看发货记录
python auto_reply.py --stats
```

### 工作流程

```
1. 顾客下单
2. 打开终端 → python auto_reply.py
3. 输入商品编号
4. 脚本生成回复文案（含链接）
5. 复制文案 → 粘贴到闲鱼 → 发送
6. 脚本自动记录发货
```

## ⚡ 推荐方案

| 阶段 | 使用方案 | 原因 |
|------|----------|------|
| 日单 < 10 | 方案一（闲鱼内置） | 够用，最简单 |
| 日单 10-30 | 方案一 + 方案二辅助 | 需要管理多个链接 |
| 日单 > 30 | 方案二为主 | 需要系统和统计 |

---

## ⚠️ 注意事项

1. 网盘链接定期检查（每周一次），失效及时更新
2. 不要在自动回复里写"永久有效"（如果链接失效会被差评）
3. 建议用阿里云盘/夸克网盘（下载不限速，用户体验好）
4. 闲鱼内置自动回复有字数限制（约 200 字），注意精简
