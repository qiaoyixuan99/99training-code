# 03-内容制作指南

> 这 10 个商品的内容，你不需要全部从零写。利用你的编程经验和 AI 工具，每人每天可以产出 2-3 个完整产品。

---

## 🛠️ 制作工具推荐

| 用途 | 工具 | 费用 |
|------|------|------|
| 写文字教程 | DeepSeek / Claude / ChatGPT | 免费 |
| 做截图标注 | 微信截图 + 系统画图工具 | 免费 |
| 制作封面图 | Canva / 稿定设计 | 免费 |
| PDF 排版 | Typora → 导出 PDF / WPS | 免费 |
| 代码编写 | Cursor / VS Code | 免费 |
| 视频录制 | OBS Studio | 免费 |

---

## 📐 制作流程（每个商品耗时约 2-4 小时）

```
步骤1: 阅读内容大纲（20分钟）
  └→ 打开 内容大纲.md，找到对应商品

步骤2: AI辅助生成初稿（30分钟）
  └→ 把大纲丢给 DeepSeek，让它生成各章节内容

步骤3: 人工修稿 + 补充截图（60-120分钟）
  └→ 补充实操截图、修正AI错误、加入个人经验

步骤4: 准备附件资料（30分钟）
  └→ 如果有代码/模板/配置，打包整理

步骤5: 排版 + 导出PDF（30分钟）
  └→ Typora排版 → 导出PDF → 上传网盘
```
