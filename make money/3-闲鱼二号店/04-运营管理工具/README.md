# 04-运营管理工具

> 两个 Python 脚本，帮你把闲鱼运营变成有节奏、可追踪的系统化操作。

---

## 📂 文件说明

| 文件 | 用途 | 使用频率 |
|------|------|----------|
| `daily_tracker.py` | 每日运营检查清单 | 每天运行一次 |
| `sales_analyzer.py` | 销售数据统计与分析 | 每周运行一次 |

---

## 🚀 快速开始

```bash
# 安装依赖（两个脚本都只用 Python 标准库，无需额外安装）
# 直接运行即可

# 每日打卡
python daily_tracker.py

# 每周分析
python sales_analyzer.py
```

---

## 📋 daily_tracker.py —— 每日运营清单

运行后会引导你完成每日运营任务：

```
✅ 任务1：擦亮所有商品
✅ 任务2：回复未读消息
✅ 任务3：上架 2-3 个新品
✅ 任务4：检查数据（曝光/点击/转化）
✅ 任务5：下架 7 天零曝光商品
✅ 任务6：检查网盘链接有效性
```

每次运行会生成一条带时间戳的打卡记录，保存在 `daily_log.txt`。

---

## 📊 sales_analyzer.py —— 销售分析

输入你的销售数据，自动计算：

```
📦 总发货数
💰 总收入
📈 日均收入
🏆 销量TOP3商品
📉 零销量商品（考虑下架）
🎯 离1000元目标还有多远
📅 趋势判断（收入在上升还是下降）
```

数据保存在 `sales_data.json`，支持逐步录入。

---

## 🔄 推荐工作流

```
每天 18:00  → python daily_tracker.py （10分钟完成日常维护）
每周日 20:00 → python sales_analyzer.py （15分钟复盘+调整策略）
每月 1号    → 月度总结，决定下月方向
```
