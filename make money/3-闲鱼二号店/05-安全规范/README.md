# 05-安全规范

> 多账号运营的安全底线。**账号安全 > 赚钱速度**，一个操作失误可能导致所有账号被连坐封禁。

---

## ⚠️ 三条不可逾越的红线

```
① 一机一号    —— 每台手机只登录一个闲鱼账号
② 一网一号    —— 每个账号用独立的手机流量（不共用WiFi）
③ 一人一号    —— 每个账号用不同身份证实名认证
```

违反任何一条 → 关联封号 → 所有账号同时被封 → 申诉成功率几乎为 0。

## 🚨 快速自检清单

每次运营前，花 30 秒确认：

```
□ 当前设备只登录了 1 个闲鱼账号？
□ 当前使用的是手机流量（非 WiFi）？
□ 这个账号的实名信息和其他账号不同？
□ 头像/昵称/简介和另一个账号完全不同？
□ 没有在文案里写"永久""破解""破解版"？
□ 虚拟商品的交付方式是网盘链接（不是直接传文件）？
```

全部打勾才能开始运营。
