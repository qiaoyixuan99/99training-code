# 闲鱼二号店 · 养号与防封完整规范

---

## 第一部分：养号流程（前 3 天）

### 为什么需要养号？

新注册的账号在闲鱼系统里是"白号"——没有行为数据，没有信用背书。如果上来就大量发布商品，系统会判定为营销号，直接限流或封禁。

### 养号三步走

#### 第 1 天：身份建设

```
☐ 用家人的身份证 + 手机号注册新的闲鱼账号
☐ 绑定支付宝（必须和闲鱼注册身份一致）
☐ 完善个人资料：
   ☐ 头像：用真实生活照（家人照片/宠物/风景），禁止用网图
   ☐ 昵称：接地气（如"小王家的小店铺"），不要带"科技""AI""专业"等词
   ☐ 简介：日常化（如"家里不用的东西，便宜出了"）
   ☐ 性别/地区/生日：如实填写
☐ 芝麻信用授权（650+ 最佳，低于 600 需要养更久）
```

#### 第 2 天：行为模拟

```
☐ 正常浏览 30 分钟以上（不要只刷不点，要有停顿、有划回）
☐ 搜索你要卖的品类关键词（如"AI教程""Python脚本"）
☐ 收藏 10-15 个同类商品（模拟真实用户比价）
☐ 浏览后点赞 5-8 个商品
☐ 给 2-3 个商品留言（如"还在吗""能便宜点吗"）
☐ 进入 3-5 个卖家的主页看看（模拟查看信誉）
```

#### 第 3 天：社交激活

```
☐ 小额购买 1-2 单（5-10 元的商品，如虚拟资料/贴纸等）
☐ 收到货后确认收货 + 给好评
☐ 发布 1-2 个"真闲置"商品（家里的旧物，不是为了卖，是为了养号）
   例：旧书、不用的杯子、孩子穿小的衣服
☐ 继续浏览同类商品 + 收藏
```

### 养号完成判断标准

| 信号 | 说明 |
|------|------|
| 发布的测试商品曝光 > 100 | ✅ 账号权重正常 |
| 搜索自己的商品能搜到 | ✅ 没有被降权 |
| 系统没有发"违规提醒" | ✅ 行为模式通过检测 |

## 第二部分：多账号隔离（核心）

### 一号店 vs 二号店配置对比表

| 配置项 | 一号店（已有） | 二号店（新开） |
|--------|---------------|---------------|
| 手机设备 | 手机 A | **手机 B**（必须是另一台） |
| 手机号 | 你的号码 A | **家人号码 B** |
| 网络 | 手机 A 的流量卡 | **手机 B 的流量卡**（不同运营商更佳） |
| 支付宝实名 | 你的身份证 | **家人的身份证** |
| Wi-Fi | 不共用 | **不共用** |
| 蓝牙 | 关闭 | 关闭 |
| GPS 定位 | 正常使用 | 正常使用（没事） |

### 绝对禁止的操作（关联封号 100%）

```
❌ 同一台手机切换登录两个账号
❌ 同一台手机安装闲鱼分身/双开软件
❌ 两个账号连接同一个 WiFi
❌ 两个账号用同一个支付宝收款
❌ 两个账号的发货地址完全一样
❌ 两个账号的退货地址完全一样
❌ 两个账号互相刷单/刷好评
❌ 用电脑模拟器登录多个闲鱼账号（可能被检测）
```

### 允许的操作

```
✅ 两台手机放在同一个房间（基站定位不同不足以关联）
✅ 同一个收货地址（但不要两个号所有地址都一样）
✅ 同一个电脑管理两个号的网盘（网盘与闲鱼无关）
✅ 用同一个电脑写文案、做资料（内容制作无关账号）
```

## 第三部分：违规红线清单

### ❌ 直接封号级别（一次就死）

| 行为 | 后果 |
|------|------|
| 售卖盗版/破解软件 | 封号 |
| 售卖色情/擦边内容 | 封号 |
| 售卖违禁品（烟草/药品/管制刀具） | 封号 + 可能追责 |
| 引导站外交易（发微信号/QQ号/手机号在描述或聊天中） | 封号 |
| 虚假发货（填假单号） | 封号 |
| 刷单/刷好评/刷曝光 | 封号 |
| 频繁发送相同消息（被判定群发广告） | 封号 |

### ⚠️ 降权级别（限流但不封号）

| 行为 | 后果 |
|------|------|
| 标题和描述不一致 | 降权 |
| 标低价引流（标 1 元实际不卖） | 降权 |
| 堆砌不相关关键词 | 降权 |
| 重复铺货（同一个商品上多个链接） | 降权 |
| 回复率过低（30% 以下） | 降权 |
| 退款率过高（10% 以上） | 降权 |
| 频繁修改商品信息（一天改好几次） | 降权 |
| 长时间不登录（超过 7 天） | 降权 |

### 虚拟商品特别注意

```
⚠️ 虚拟商品在闲鱼属于"灰色地带"：
  - 平台不禁止但也不鼓励
  - 容易出现"虚假发货"纠纷（因为没物流单号）
  - 售后纠纷中平台倾向偏向买家

✅ 降低虚拟商品风险的策略：
  1. 标题加"资料整理服务""代办指导服务"（不要只写"卖资料"）
  2. 发货时在聊天窗口再发一次链接（留记录）
  3. 考虑附带一个实体物品（如一张打印的二维码卡片）寄出去
  4. 商品描述最后加"虚拟商品发货后不退不换"
  5. 开通"24小时极速退款"（减少纠纷率）
```

## 第四部分：违规词黑名单

### 标题/描述中永远不要出现的词

```
永久      → 改成"长期"
破解/破解版 → 绝对不用
盗版      → 绝对不用
免费      → 改成"分享""整理"
全网最低   → 改成"超值"
最        → 广告法禁词，不用
第一      → 同上
正版（没授权时）→ 不用
官方授权（没授权时）→ 不用
微信号/QQ号/手机号 → 绝对不写在描述里
加微信    → 绝对不写
```

### 聊天中不能发的

```
微信号
QQ号
手机号
二维码图片
站外链接（淘宝/拼多多/其他平台）
"加我微信聊"
"点击链接购买"
```

> ⚠️ 闲鱼会自动扫描聊天内容，检测到联系方式会直接警告甚至封号。

## 第五部分：出问题怎么办

### 常见问题处理

| 问题 | 处理方式 |
|------|----------|
| 商品被下架 | 查看违规原因 → 修改文案 → 重新上架 |
| 收到警告通知 | 立即停止当前操作 → 检查文案是否有违禁词 |
| 账号被限流 | 停止发品 3 天 → 只做浏览行为 → 再慢慢恢复 |
| 账号被封 | 申诉（成功率低）→ 总结经验 → 用新身份重新开始 |

### 申诉模板（如果被封）

```
尊敬的小二：

我的账号[账号名]被限制发布/登录，我仔细查看了平台规则，
认为可能存在误判。

我发布的是自己整理的教程资料，内容完全原创，不涉及任何
盗版或侵权内容。如果存在不合规的地方，请告知具体原因，
我会立即整改。

作为闲鱼多年的忠实用户，我一直遵守平台规则，希望平台能
给一次机会，谢谢。
```

## 第六部分：长期安全运营守则

### 每日
```
☐ 不主动在聊天中发站外联系方式
☐ 新上架的商品检查标题有没有违禁词
☐ 虚拟商品发货后在聊天窗发送链接（保留记录）
```

### 每周
```
☐ 检查所有在售商品，下架超过 7 天零曝光的品
☐ 检查是否有买家投诉/差评，主动联系解决
☐ 确认网盘链接没有失效
```

### 每月
```
☐ 检查两个账号的隔离是否完好（没有串号使用）
☐ 复盘本月违规/警告记录
☐ 关注闲鱼规则更新（平台政策可能变化）
```

## 💡 核心原则总结

```
1. 安全第一          账号没了什么都没了，不要为了一点利润违规
2. 隔离到底          两个号在设备/网络/身份三个维度完全独立
3. 慢慢来比较快      养号急不得，权重是积累出来的
4. 内容原创          所有教程资料必须自己整理/编写
5. 服务到位          好评率是最大的护城河
```
