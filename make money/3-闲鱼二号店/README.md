# 闲鱼二号店 · AI工具教程 + 虚拟资料矩阵

> **一句话定位**：把编程/AI/自动化领域的知识，包装成标准化数字产品，一份资料卖 N 次。
>
> **目标**：月收入 1000 元（日均 3-4 单，客单价 9.9 元）
>
> **与一号店关系**：一号店卖定制服务（高客单·高沟通），二号店卖标准化资料（低客单·零边际成本），两者互补不抢客。

## 🎯 1000 元目标拆解

| 阶段 | 时间 | 核心动作 | 预期日收入 |
|------|------|----------|------------|
| **① 养号期** | 第 1-3 天 | 模拟真实用户，浏览/收藏/小额购买 | 0 元 |
| **② 铺货期** | 第 4-14 天 | 每天上架 3-5 个品，累计 30+ 链接 | 10-20 元 |
| **③ 出单期** | 第 15-30 天 | 筛选爆款，优化标题和首图 | 30-50 元 |
| **④ 稳定期** | 第 2 个月起 | 保留高转化链接，日常维护 | 33+ 元 |

```
客单价 9.9 元 × 日均 3.4 单 × 30 天 = 1009 元 / 月 ✅
```

## ⚡ 快速开始（5 分钟上手）

### 第一步：选择第一个上架商品
进入 `01-商品文案库/`，选择 **"DeepSeek保姆级教程"**（需求最大，出单最快）

### 第二步：准备内容
对照 `03-内容制作指南/内容大纲.md`，创建对应的 PDF 资料文件

### 第三步：配置自动发货
进入 `02-自动发货系统/`，按照 README 配置 Python 自动回复脚本

### 第四步：上架
复制商品文案 → 粘贴到闲鱼 → 设置自动回复 → 开始出单

### 第五步：每日维护
运行 `04-运营管理工具/daily_tracker.py` 跟踪每日任务

## ⚠️ 核心原则

1. **一机一号**：两个闲鱼账号必须在不同手机/网络上运行
2. **坚持日更**：每天都要擦亮+上新，断更权重会明显下降
3. **数据驱动**：7 天零曝光的品果断下架，爆款加量复制
4. **绝不碰盗版**：所有内容必须原创或基于公开资源整理
