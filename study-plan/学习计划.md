## 一、核心理念：改变思维，而非完成任务

传统学习计划的问题是：它让你专注于「做完什么」，而非「成为什么」。  
科学研究（James Clear, BJ Fogg, Carol Dweck）一致指向同一条路：

```
身份认同 → 微行为 → 反思强化 → 身份升级
```

### 我们要做的事：

| 旧思维（清单模式） | 新思维（身份模式） |
|-------------------|-------------------|
| "我今天要背 15 个单词" | "我是一个英语使用者，今天自然要接触英语" |
| "看完 LangChain 文档第 3 章" | "我是一个 AI 工程师，今天要搞懂 RAG 原理" |
| 完成 = 打卡，没完成 = 失败 | **最低 5 分钟也算赢**，连续比完美重要 |
| 学完就完了 | **每次学习后写一句话反思**，把信息变成理解 |

### 三条铁律：

1. **永不断链两天。** 允许某天只做 5 分钟，不允许连续两天归零。断一天是意外，断两天是放弃。
2. **每次学习必须产出一句话反思。** 不写反思 = 白学。反思内容：「我刚才搞懂了什么之前不懂的？」
3. **身份检查每周一次。** 问自己：「这周，我离『AI 工程师』/『英语使用者』近了多少？」（1-10 分）

## 二、时间-能量映射

不按「什么时间做什么事」来分，按**能量状态**来分。

| 时段 | 能量类型 | 适合的学习模式 |
|------|----------|--------------|
| 上午 1h（零散） | 接收能量 | 语法学习 + 泛听（低认知负荷，可中断） |
| 中午 20min（短促） | 补充能量 | Anki 词汇突击（高密度短时记忆） |
| 下午 1h（零散） | 消化能量 | 英文技术精读 + 长难句拆解（中等负荷） |
| 晚上 1.5h（整块） | 创造能量 | Agent 编程实战（高认知负荷，需要专注） |

## 三、奖励机制设计

以下是 5 种经过科学研究验证的奖励机制。**本次在 App 中实现了方案 B + 方案 D 的组合。**

### 五种可选方案对比

| 方案 | 原理 | 驱动力 | 实现难度 | 适合人群 |
|------|------|--------|----------|----------|
| **A. 损失厌恶型** | 预存押金（如 2000 元），未完成任务则扣款捐给不喜欢的机构。损失带来的痛苦 ≈ 获得收益的 2 倍（Kahneman, 1979） | ⭐⭐⭐⭐⭐ | 需要支付系统 | 自律较弱、需要外部压力 |
| **B. 变量奖励型** 🎯 | 完成任务获得随机 XP（10% 概率触发 2-3x 暴击），不可预测性激活多巴胺回路（类似老虎机机制） | ⭐⭐⭐⭐ | 低（前端即可） | 喜欢游戏、需要新鲜感 |
| **C. 社会承诺型** | 在朋友圈/社群公开每日打卡，未完成则发红包。社会评价是强驱动力（Cialdini, 2006） | ⭐⭐⭐⭐ | 需要社交平台 | 在意他人看法、有社交圈 |
| **D. 身份晋级型** 🎯 | 经验值累积 → 等级提升 → 解锁新头衔（如「Prompt 学徒 → AI 工程师」）。进度可视化提供持续正反馈 | ⭐⭐⭐ | 低（前端即可） | 目标感强、喜欢成长叙事 |
| **E. 实物兑换型** | 积分可兑换自己设定的奖励（看剧时间、游戏时间、购物）。将延迟满足具体化 | ⭐⭐⭐ | 需要自制兑换规则 | 有明确爱好、需要具体奖励 |

### 本次实现：B（变量奖励）+ D（身份晋级）

**为什么选这个组合？**
- 变量奖励提供**短期多巴胺**（每天都有惊喜可能）
- 身份晋级提供**长期意义感**（15 个等级映射你的成长路径）
- 两者互补：B 解决「今天不想动」的问题，D 解决「学了一个月不知道有什么用」的问题
- 无需外部系统，纯前端可运行

**15 级身份晋升路径：**
```
Lv.1  Prompt 学徒      → Lv.6  Function 召唤师   → Lv.11 LLM 驯兽师
Lv.2  Grammar 工匠     → Lv.7  Agent 架构师       → Lv.12 推理系统设计者
Lv.3  RAG 探索者       → Lv.8  Multi-Agent 指挥   → Lv.13 双语 AI 先锋
Lv.4  Vector 航海家    → Lv.9  LangGraph 塑造者   → Lv.14 全栈 Agent 大师
Lv.5  Chain 编织者     → Lv.10 AI 工程师          → Lv.15 第二职业就绪 🚀
```

## 五、如何使用 Daily Forge App

### 启动方式
直接在浏览器打开 `app/index.html`，推荐手机浏览器添加到主屏幕（PWA 体验）。

### 每日流程
1. **打开 App** → 看到今天的 4 个锻造任务
2. **完成一个时段的学习** → 点击「✅ 完成打卡」
3. **写一句话反思** → App 会弹出随机反思提示（如「用一句话总结刚才学到的核心概念」）
4. **获得 XP** → 10% 概率触发 2-3x 暴击奖励
5. **全完成** → 连续天数 +1，每 7 天额外 +50 XP 奖励
6. **升级** → XP 累积达到阈值自动升级，解锁新身份头衔

### 每日和 Claude 的互动方式

每天学习结束后，跟我说你的感受。格式自由，比如：
- "今天语法单元做完了，但 Anki 只背了 8 个词，晚上 RAG 部分搞懂了 chunking"
- "今天太累了只做了最低限度，但至少没断链"

我会帮你：
1. 更新本文件中的「当前状态」和「每日记录」
2. 提炼你注意不到的进步模式
3. 在方向偏移时提醒调整
